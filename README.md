# Evolutionary Strategy Application

## How to run?
1) download repo
`git clone https://github.com/kubaunold/evolutionaryAlgorithm.git`
2) create virtual env...
`virtualenv venv`
...  and activate it
`.\venv\Scripts\activate`
3) install required packages
`pip install -r requirements.txt`
4) run app
`python app.py`
